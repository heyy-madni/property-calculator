Property Calculator (India Focused)
Overview

This is a property investment calculator I built to analyze residential rental properties in the Indian market.

The idea is simple.
You enter the financial details of a property and the program generates a report showing whether the deal makes sense financially.

I built this project mainly to practice:

Python

Financial calculations

Real estate analysis

Using Pandas for structured data

Thinking in terms of capital allocation

This is a learning project. It is not meant to be used for real investment decisions.

What It Does

You input details like:

Property price

Loan amount

EMI

Cash invested

Monthly rent

Annual maintenance

Expected appreciation rate

Rent growth rate

Vacancy assumption

From this, the calculator generates:

Monthly cash flow

Annual cash flow

Cash-on-cash return

Gross rental yield

Loan-to-value ratio

5-year projected property value

5-year projected rent

An internal investment score

At the end, it gives a simple report summarizing the financial position of the property.

Assumptions (India Context)

The calculations are based on typical Indian residential property conditions:

EMI-based home loans

Common LTV ranges used by Indian banks

Conservative appreciation assumptions

Average rental yield patterns seen in Indian cities

This is a simplified financial model.

Investment Score

The project assigns a basic score based on:

Whether cash flow is positive or negative

Rental yield strength

LTV risk

Appreciation potential

The score is only for comparison inside this model.
It is not a professional rating system.

Important

This project is only for practice.

Real-world real estate investing involves far more complexity:

Legal due diligence

Market cycles

Taxes

Inflation

Liquidity risk

Unexpected capital expenditure

Regulatory changes

This calculator ignores many of those factors.

Do not use it to make actual financial decisions.
